## Image Guidelines

- **Format:** PNG or JPG
- **Recommended size:** 800x600 pixels or similar aspect ratio
- **File naming:** Use lowercase with hyphens (already set up)
- **Quality:** Clear, professional screenshots showing your projects

## How to Add Images

1. Take screenshots of your projects
2. Rename them according to the list above
3. Drop them into this folder
4. Refresh your website at http://localhost:8080

The website will automatically show your images when they're added!
If an image is missing, it will show a colored placeholder instead.
